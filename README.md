# Library Management System

## Description
This is a MySQL-based Library Management System. It manages:

- Books
- Authors
- Categories
- Library Members
- Book Loans  

The database demonstrates proper relational design with:

- Primary keys (PK)
- Foreign keys (FK)
- Unique constraints
- Not Null constraints
- Relationships: 1-1, 1-M, M-M
